# 📊 Proyectos Power BI – Nicolás Osinaga

Colección de **dashboards desarrollados en Power BI**, enfocados en la generación de insights accionables y comunicación visual efectiva para la toma de decisiones empresariales.  

Cada proyecto combina **modelado de datos, DAX avanzado, storytelling con datos y diseño UX**, siguiendo las mejores prácticas de Microsoft Power BI.

---

## 🧠 Habilidades aplicadas

| Área | Competencias |
|------|---------------|
| 📊 **Modelado BI** | Tablas de hechos y dimensiones · Relaciones · Normalización de datos |
| ⚙️ **DAX & Time Intelligence** | Medidas dinámicas · KPIs · YTD, YoY, variaciones porcentuales |
| 🎨 **Visual Design** | UX para dashboards ejecutivos · Colorimetría corporativa · Jerarquía visual |
| 🔄 **Power Query** | Limpieza de datos · Transformaciones M · Combinaciones y merges |
| 🗃️ **Conectividad de Datos** | SQL Server · Excel · Web APIs |
